# 🌾 AgriFood Italia 2030 — Vota le Tendenze

Interfaccia interattiva per votare e classificare i trend più rilevanti per il settore agroalimentare italiano, suddivisi in 5 categorie: **Tecnologico**, **Demografico**, **Sociale**, **Ambientale** e **Normativo**.

🔗 **Demo live:** [eleonorabarelli.github.io/agrifood-trends-italia](https://eleonorabarelli.github.io/agrifood-trends-italia/)

---

## Funzionalità

- **20 tendenze** selezionate tra i principali driver di cambiamento per l'agrifood italiano
- **Voto per rilevanza** — ogni utente può votare (e revocare il voto) su ciascuna tendenza
- **Classifica live** — leaderboard aggiornata in tempo reale con barre di avanzamento
- **Filtro per categoria** — esplora le tendenze per area tematica
- **Persistenza locale** — i voti vengono salvati nel browser (localStorage)
- **Design responsive** — ottimizzato per desktop, tablet e mobile

## Categorie

| Categoria | Descrizione |
|-----------|-------------|
| 💡 Tecnologico | AI, IoT, blockchain, robotica, vertical farming |
| 👥 Demografico | Invecchiamento, neo-rurali, manodopera stagionale |
| 🤝 Sociale | Plant-based, turismo enogastronomico, km0, spreco alimentare |
| 🌱 Ambientale | Siccità, biologico, agroecologia, energie rinnovabili |
| ⚖️ Normativo | Farm to Fork, GDO, internazionalizzazione, novel food |

## Tecnologie

- HTML5 / CSS3 / JavaScript vanilla — zero dipendenze
- Google Fonts (Playfair Display)
- localStorage per la persistenza dei voti

## Utilizzo locale

Clona il repository e apri `index.html` direttamente nel browser — nessun server necessario.

```bash
git clone https://github.com/eleonorabarelli/agrifood-trends-italia.git
cd agrifood-trends-italia
open index.html   # macOS
start index.html  # Windows
```
